# Change Management System

A web-based system for managing change requests in software projects.

## Features
- User authentication with JWT
- Change Request management  
- Role-based access control
- RESTful API
- CI/CD Pipeline

## Installation
npm install
npm start

## Testing
npm test
npm run coverage

## CI/CD
This project includes a complete CI/CD pipeline with:
- Automated testing
- Code coverage
- Security scanning
- Deployment artifacts
